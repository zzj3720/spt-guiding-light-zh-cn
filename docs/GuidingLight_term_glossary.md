## Tone rules

- Trader text can be sharp, sarcastic, or rough, but avoid internet slang.
- Keep item names and boss names in the form Tarkov players actually use.
- If a line mixes Chinese and English, it should be because the English term is intentionally preserved, not because translation was left unfinished.
- Prefer one stable wording over a “better sounding” variant if both mean the same thing.

## Explicit fixes to enforce

- Never leave `stab` untranslated; use `突刺`.
- Never leave `('black)'` untranslated; render it as `黑肢`.
- Do not use `5.56x54mm NATO`; use `5.56x45mm NATO`.
- Prefer `信任度` over `忠诚度` for EFT loyalty terminology.
- Avoid `打真人`; prefer `跟真人交手` or `对付真人`.

# 过程文档

这个目录保留了本次汉化整理过程中最有价值的中间记录，方便后续继续维护。

说明：

- 这里出现的 `manual / 手工 / 手动` 表述，指的是 `GPT-5.4 / Codex` 逐条手动执行的审校过程。
- 这不是“人类译者逐条校对”的意思。

建议优先看这些文件：

- [GuidingLight_translation_notes.md](./GuidingLight_translation_notes.md)
- [guidinglight_review_sheet.tsv](./guidinglight_review_sheet.tsv)
- [guidinglight_review_summary.json](./guidinglight_review_summary.json)
- [GuidingLight_manual_review_progress.md](../GuidingLight_manual_review_progress.md)

文件说明：

- `guidinglight_review_sheet.tsv`: 全量 review sheet，包含英文、中文和审校基础信息
- `guidinglight_review_sheet_ascii.tsv`: 便于编码环境较差时查看的 ASCII 版本
- `guidinglight_review_*`: 阶段性逐条手动审阅记录
- `GuidingLight_translation_notes.md`: 最早期的范围定义和术语约束说明
